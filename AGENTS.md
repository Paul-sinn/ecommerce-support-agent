# 프로젝트 작업 규칙

## 승인 규칙
- 먼저 원인과 수정 계획을 설명한다.
- 사용자 승인 전에는 코드 수정하지 않는다.
- 사용자 승인 전에는 패키지 설치, 버전업, 테스트 실행하지 않는다.
- 승인 후에만 변경을 수행한다.

## 프로젝트 엔트리포인트 메모
- `main.py`: CLI entrypoint
- `lang_cli.py`: `langgraph dev` entrypoint
- `langgraph.json`: LangGraph 설정 파일

## 금지 사항
- 승인 전 파일 수정 금지
- 승인 전 패키지 설치/버전업/테스트 금지

## 응답 방식
- 먼저 원인과 계획을 설명한다.
- 승인 후에만 실제 실행과 수정을 진행한다.
